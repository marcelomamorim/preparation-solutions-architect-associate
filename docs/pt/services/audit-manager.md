## Tópicos Principais

### O que é o AWS Audit Manager?

Bem -vindo ao usuário do AWS Audit Manager
Guia.

O AWS Audit Manager ajuda você a auditar continuamente seu uso da AWS para simplificar como você gerencia riscos e
conformidade com regulamentos e padrões do setor.O gerenciador de auditoria automatiza a coleção de evidências para que você
pode avaliar mais facilmente se suas políticas, procedimentos e atividades também conhecidas como controles estão operando efetivamente.Quando chegar a hora de um gerente de auditoria, o gerente de auditoria
Ajuda você a gerenciar análises das partes interessadas de seus controles.Isso significa que você pode construir pronto para a auditoria
relatórios com muito menos esforço manual.

O Audit Manager fornece estruturas pré -construídas que estruturam e automatizam avaliações para um determinado
padrão de conformidade ou regulamentação.As estruturas incluem uma coleção pré -construída de controles com
Descrições e procedimentos de teste.Esses controles são agrupados de acordo com os requisitos de
o padrão de conformidade ou regulamento especificado.Você também pode personalizar estruturas e controles
Para suportar auditorias internas de acordo com seus requisitos específicos.

Você pode criar uma avaliação a partir de qualquer estrutura.Quando você cria uma avaliação, gerente de auditoria
executa automaticamente as avaliações de recursos.Essas avaliações coletam dados para as contas da AWS
que você define como em escopo para sua auditoria.Os dados coletados são automaticamente
transformado em evidências favoráveis ​​à auditoria.Então, está ligado aos controles relevantes para ajudar
Você demonstra conformidade em segurança, gerenciamento de mudanças, continuidade de negócios e software
licenciamento.Este processo de coleta de evidências está em andamento e começa quando você cria seu
avaliação.Depois de concluir uma auditoria e você não precisa mais do gerente de auditoria para coletar evidências, você pode
Stop Evidence Collection.Para fazer isso, altere o status da sua avaliação para inativo.
