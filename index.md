# AWS Solutions Architect Associate Exam Guide

Welcome to the AWS Solutions Architect Associate Exam Guide. This site provides comprehensive resources and structured content to help you prepare for the AWS Solutions Architect Associate Certification. You can navigate through the course materials in both English and Portuguese by selecting the desired language below.

## Site Overview

This guide is divided into the following main sections:

- **Introduction**
  - Overview of the certification, its importance, and exam details.
- **Domain 1: Design Resilient Architectures (30%)**
  - Learn about multi-tier architecture solutions, high availability, fault tolerance, and decoupling mechanisms.
- **Domain 2: Design High-Performing Architectures (28%)**
  - Explore elastic and scalable compute solutions, high-performing storage solutions, networking solutions, and database solutions.
- **Domain 3: Design Secure Applications and Architectures (24%)**
  - Understand how to design secure access to AWS resources, secure application tiers, and appropriate data security options.
- **Domain 4: Design Cost-Optimized Architectures (18%)**
  - Identify cost-effective storage, compute, and database services, and design cost-optimized network architectures.
- **Additional Resources**
  - Access study materials, practice exams, AWS whitepapers, and other helpful links.
- **Conclusion**
  - Final tips, how to schedule your exam, and a recap of key points.
